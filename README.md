# MNIST Digit Classifier: Perceptron vs ANN vs CNN
This project demonstrates the implementation and comparison of three neural network architectures — Perceptron, Artificial Neural Network (ANN), and Convolutional Neural Network (CNN) — on the MNIST handwritten digit dataset.

## Features
- Pixel normalization and image reshaping
- One-hot encoding of labels
- Perceptron, ANN, and CNN model implementation
- Training history visualization (accuracy & loss)
- Confusion matrix for CNN
- Side-by-side prediction comparison across all three models
- Final accuracy bar chart comparison

## Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Models Built
- **Perceptron** — Single-layer linear classifier
- **ANN** — Multi-layer fully connected network with ReLU activations
- **CNN** — Convolutional network with pooling and dropout layers

## Learning Outcome
- Understood how different neural network architectures majorly **CNN** perform on image data
- Learned the impact of convolution layers on spatial feature extraction
- Practiced model evaluation using accuracy, loss curves, and confusion matrices
- Compared Perceptron, ANN, and CNN performance on the same dataset
